Page headers, footers, numbers, TOC, using CSS Regions

Steps:

  1. Download a recent version of Chromium 22.0.1215.0+
  2. In Chrome/Chromium open a new tab and go to "chrome://flags/"
  3. Search for "CSS Regions" and enable them
  4. Restart your browser
  5. Read the "Notice" below so you know what to look for and then
  6. Check out the Editor Example Page
  7. or the Book Example Page

Notice:

  1. There are multiple numbered "pages"
  2. The TOC is generated from the H1 tags and includes the page number the tag occurs on
  3. To reduce time spent reflowing text every time a page is added, batch pages are added and removed (300 pages in ~7min)
